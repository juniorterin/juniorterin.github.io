Gerência de Marketing Layout criado pela Anna Kelmer para uma empresa de assessoria de marketing. Com embasamento em questões técnicas sobre programação, se trata de um site extremamente simples, um one-page de puro HTML provando que nem sempre é necessário WordPress para exibir um conteúdo estático, simples e objetivo, visando também não agregar custos ao desenvolvimento do projeto. Speed Brasil Template para WordPress responsivo e leve feito com a ajuda do Bones. O cliente precisava de um pequeno sistema de cadastro, consulta e download de notas fiscais. Para executar o trabalho usei o Advanced Custom Fields para gerar as entradas personalizadas contendo os dados que demandava o trabalho, e para a importação dos dados (mensalmente eram mais de 30000 entradas) foi usado o plugin WP All Import que diariamente em um horário agendado ou por meio de um gatilho criado nos servidores, fazia toda a importação dos dados que entravam por meio de um CSV gerado pelo sistema interno da empresa. O layout foi desenvolvido pela equipe de criação da Agência Aktus de Vinhedo, empresa da qual trabalhei aproximadamente 1 ano de 2014 até 2015. Conheça o site clicando aqui.
